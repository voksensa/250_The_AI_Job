# Gate G1: Research & Problem Definition

## Purpose
Ensure every significant task is grounded in up-to-date research, competitive analysis, or user insight—not guesswork.

## Required Evidence Files

- `TASK-{TASK_ID}-research-report.md` - Summary of problem, current alternatives, and references
- `TASK-{TASK_ID}-sources.json` - JSON list of sources with metadata

## Standards

- At least **3 independent, authoritative sources** (docs, research, reputable blogs, news)
- For market/tech claims, at least **1 primary source** (official doc or vendor)

## CEO Checklist

- [ ] Research report exists and is non-empty
- [ ] Sources.json is valid JSON
- [ ] Sources.json contains ≥3 entries
- [ ] At least 1 primary source cited for technical claims
- [ ] Research summary aligns with cited sources (manual check)

## Template Files

- `TEMPLATE-research-report.md` - Report structure
- `TEMPLATE-sources.json` - Sources JSON schema
